sequenceDiagram

    CA->>SB: update session
    CA-->>FE: agent message + suggestions
  end

  U->>FE: complete
  FE->>O: POST /api/project/start

  par Wave 1

    R->>R: web_search regulatory updates
    R-->>O: regulatory updates

    O->>O: build inboxes for specialists

Knowledge Base (Static)
